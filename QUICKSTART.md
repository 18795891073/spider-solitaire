# 快速开始指南

## 安装和运行

1. **安装依赖**
   ```bash
   npm install
   ```

2. **启动开发服务器**
   ```bash
   npm run dev
   ```

3. **打开浏览器**
   访问控制台显示的本地地址（通常是 http://localhost:5173）

## 游戏操作

### 基本操作
- **移动卡片**: 点击并拖拽卡片到目标位置
- **双击卡片**: 自动尝试移动到合适的位置
- **发牌**: 点击右上角的发牌堆
- **撤销**: 点击"撤销"按钮（最多50步）

### 游戏提示
- 点击"提示"按钮，系统会高亮显示可移动的卡片（持续3秒）

### 游戏设置
- **难度选择**: 可以选择单花色、双花色或四花色
- **自动完成**: 勾选后，完整的K到A序列会自动移动到完成区

## 游戏规则

### 移动规则
- 只能移动正面朝上的牌
- 可以移动单张牌或同花色降序序列（K→Q→J...→A）
- 空列只能放置K或K开始的序列
- 不同难度下对花色要求不同：
  - 单花色：必须同花色
  - 双花色：同一颜色即可
  - 四花色：只要点数匹配即可

### 胜利条件
- 当8个完成区都有完整的同花色K到A序列时，游戏胜利

## 开发说明

项目使用 Vue 3 Composition API + TypeScript 开发，代码结构清晰：

- `src/components/` - Vue组件
- `src/composables/` - 组合式函数（游戏逻辑、拖拽等）
- `src/types/` - TypeScript类型定义
- `src/utils/` - 工具函数

## 构建生产版本

```bash
npm run build
```

构建后的文件在 `dist/` 目录中。
